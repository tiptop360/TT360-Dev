## Task 3: Compress Hero Images

**Where:** [tinypng.com](https://tinypng.com) (free, no signup)

**Steps:**
1. Go to Shopify Admin → Content → Files
2. Download your 3–5 biggest homepage images (hero banner, product shots)
3. Drag each into TinyPNG.com
4. Download the compressed versions
5. Re-upload to Shopify Files
6. In Theme Editor, replace old images with new compressed ones

**Target:** Each image under 200KB. Hero banner under 500KB.

**Alternative:** Install Shopify app **"Crush.pics"** — set to auto-compress all future uploads.

---

## Task 4: Add Trust Badges Section

**Where:** Shopify Theme Editor → Homepage

**Steps:**
1. Click "Add section" → choose "Multi-column" or "Image with text"
2. Add 5 small badge icons (80×80px PNG, transparent background)
3. Use Canva free to make them, or download free icons from [flaticon.com](https://flaticon.com)

**Badge text to display:**
- 🔒 Secure Payment
- 🚚 Free Next-Day Delivery UAE
- ↩️ Easy 7-Day Returns
- ⭐ 4.8/5 from 189 Parents
- 🇦🇪 UAE-Based Support

**Placement:** Directly below your main hero banner, before product listings.
